# DSA & LeetCode Practice

This repository contains my daily **Data Structures and Algorithms (DSA)** and **LeetCode** practice solutions.

The purpose of this repository is to maintain consistency, track my learning journey, and document the problems I solve over time. Each day's practice is organized into its own folder (e.g., `Day 1`, `Day 2`, `Day 3`, ...), making it easy to follow my progress.

## Repository Structure

```text
Day 1/
Day 2/
Day 3/
...
Day N/
```

Each folder contains the solutions completed on that particular day.

## Goals

* Practice DSA consistently.
* Improve problem-solving skills.
* Build a strong foundation for coding interviews.
* Keep a record of daily progress.

This repository will continue to grow as I solve more problems and learn new concepts.
